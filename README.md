# Disney+ Clone Coding Project

이 프로젝트는 Disney+의 UI를 클론하여 만든 React 기반의 웹 어플리케이션입니다. Firebase를 사용하여 데이터를 관리하고, React.js, Axios, React Router, 그리고 Styled Components를 활용하여 구현되었습니다.

## 기술 스택

- React.js
- Axios
- React Router
- Styled Components
- Firebase

## 빌드 방법

- npm install
- npm start
- npm run build
